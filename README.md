# Vendor and Order Tracker

#### Created by Christen Weston

#### This project lets you track vendors and orders

## Technologies Used

* Git
* C#
* .NET 5.0
* MS Test
* MVC
* Razor

## Setup Installation Requirements

1. Clone the Vendor and Order Tracker repository
2. Open in your favorite IDE
3. CD VendorAndOrderTracker.Tests run "dotnet restore"
4. run "dotnet test" to test
5. CD ..
6. CD into VendorAndOrderTracker
7. run "dotnet build"
8. run "dotnet run" to use the program
9. Enjoy!
